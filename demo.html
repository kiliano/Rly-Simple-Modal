Configuração Avançada

Podemos ainda alterar algumas configurações do modal para aproveita-lo melhor.

  1. Por padrão, o modal está programado para salvar no local storage sempre que o usuário clicar no botão de fechar ou dentro de algum link. Isso faz com que o popup seja aberto apenas 1 vez. Para que o modal seja exibido sempre que o usuário entre na página, basta remover o conteúdo do atributo "popupsalvar".
  2. Caso você queira tirar o Popup do ar ou faze-lo não ser exibido automaticamente, basta mudar o atributo "popupabrir" de sim para nao.
  3. Você pode escolher abrir o modal com uma chamada, em vez de faze-lo abrir automaticamente. Experimente clicando aqui. Para fazer esse efeito, basta colocar o href de algum link apontando para a âncora do ID do modal que foi criado (nesse caso, href="#rly-popup-id-0")
