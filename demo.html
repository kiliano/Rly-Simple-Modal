Configs

We can also change some modal settings fot better experience.

  1. By default, the modal save on the local storage whenever the user clicks the close button or within a link. This causes the popup to open only once. For the modal to be displayed whenever the user enters the page, simply remove the contents of the "popupsave" attribute.
  2. If you want Popup don't show automatically, simply change the attribute "popupabrir" to nope
  3. You can choose to open the modal with a call instead of having it open automatically. Try click here. To do that just create a "a" and put the ID of modal in href. Like: href="#rly-popup-id-0"
